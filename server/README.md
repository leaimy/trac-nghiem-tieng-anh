## 2. Cách cài đặt Server

- Tải dự án về
- Đổi tên tập tin `ninja-config-example.json` thành `ninja-config.json`
- Mở terminal, di chuyển vào thư mục `server` và chạy lệnh `php -S 127.0.0.1:5000 -t public_html`
